# Demo Photo Prompts — Home Page Film Strip

Generated by /plan-design-review on 2026-03-21

## Usage

These prompts are for generating the 4 before/after photo pairs used in the
home page film strip. Each pair requires two images: a degraded "before" and a
restored "after" representing the same subject.

**Render specs:**
- Aspect ratio: **2:3 portrait** (width × height)
- Recommended render size: **800 × 1200px**
- Format: JPG at 85% quality (~150–200KB per image)
- Color: Before images should be B&W or faded early color; After images should
  be warm, clear, restored (B&W or gentle warm tone — not over-saturated)

**Where to save the rendered images:**

```
public/
  demo/
    pair-1-before.jpg   ← grandmother portrait, before
    pair-1-after.jpg    ← grandmother portrait, after
    pair-2-before.jpg   ← wedding couple, before
    pair-2-after.jpg    ← wedding couple, after
    pair-3-before.jpg   ← family outdoor gathering, before
    pair-3-after.jpg    ← family outdoor gathering, after
    pair-4-before.jpg   ← child school portrait, before
    pair-4-after.jpg    ← child school portrait, after
```

## Notes for rendering

- If using **Midjourney**: append `--ar 2:3 --style raw --v 6` to each prompt
- If using **DALL-E 3**: the aspect ratio prompt instruction is usually sufficient;
  request "portrait orientation"
- If using **Stable Diffusion**: add `--width 800 --height 1200` in your generation settings
- **Critical:** The before/after pairs must look like they're the same photo
  (same subject, same pose, same setting). If your AI tool struggles to match them,
  generate the "after" first, then use img2img with heavy degradation filters to
  produce the "before" — this guarantees visual consistency.
- **Warm tones only:** Per DESIGN.md, never cold or clinical imagery. Review renders
  and reject any that feel blue, grey, or sterile.
